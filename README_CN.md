##  快速开始

### 方式一：在线使用（推荐）⭐

直接访问 [在线演示](https://1822520752.github.io/badmintonrapidscoreboard/) 即可开始使用，无需下载。

### 方式二：本地文件

```bash
# 1. 下载 index.html 文件
# 2. 双击在浏览器中打开
# 3. 开始使用！
```

### 方式三：克隆仓库

```bash
# 克隆仓库
git clone https://github.com/1822520752/badmintonrapidscoreboard.git

# 进入目录
cd badmintonrapidscoreboard

# 方式A：直接打开
# 双击 index.html 文件

# 方式B：Python服务器（推荐）
python -m http.server 8000
# 访问 http://localhost:8000

# 方式C：Node.js服务器
npx serve .
# 访问 http://localhost:3000

# 方式D：VS Code Live Server
# 安装 Live Server 扩展，右键 index.html 选择 "Open with Live Server"
```

### 方式四：GitHub Pages

1. Fork 本仓库
2. 进入 Settings → Pages
3. 选择 `main` 分支作为 Source
4. 保存后通过 `https://你的用户名.github.io/badmintonrapidscoreboard/` 访问

## ❓ 常见问题

<details>
<summary><b>🔧 安装和使用</b></summary>

**问：需要安装什么吗？**  
答：不需要！这是一个纯前端应用。只需在浏览器中打开 `index.html` 或访问在线演示即可。

**问：支持哪些浏览器？**  
答：支持所有现代浏览器：
- ✅ Chrome（推荐）
- ✅ Firefox
- ✅ Safari
- ✅ Edge
- ❌ IE（不支持）

**问：可以在手机上使用吗？**  
答：当然可以！本应用采用响应式设计，完美支持手机、平板和所有设备。

**问：需要联网吗？**  
答：不需要！本应用完全离线可用。所有功能无需网络连接。

</details>

<details>
<summary><b>💾 数据相关问题</b></summary>

**问：我的数据存储在哪里？**  
答：数据存储在浏览器的 LocalStorage 中，仅在本地设备上，从不上传到任何服务器。

**问：我的数据会丢失吗？**  
答：正常使用不会导致数据丢失。但以下情况会删除数据：
- 清除浏览器缓存/数据
- 使用隐私/无痕模式
- 卸载浏览器  
建议定期备份数据。

**问：如何备份我的数据？**  
答：进入"设置" → "数据管理" → "备份数据"，会下载一个JSON文件。请妥善保存。

**问：如何恢复我的数据？**  
答：进入"设置" → "数据管理" → "恢复数据"，选择之前导出的JSON文件。

</details>

<details>
<summary><b>🎮 功能使用问题</b></summary>

**问：如何更改目标分数？**  
答：开始比赛前，点击"设置"按钮可自定义目标分数（默认为21）。

**问：如何撤销误操作？**  
答：点击"-"按钮可撤销上一次计分操作。支持多次撤销。

**问：分组后可以重新分组吗？**  
答：可以！点击"重新分组"按钮可生成新的比赛组合。

**问：成就是如何解锁的？**  
答：满足条件时成就会自动解锁。解锁时会弹出提示。

</details>
