## 🚀 快速开始

### 在线使用（推荐）

直接访问 **[在线演示](https://1822520752.github.io/badmintonrapidscoreboard/)** 即可开始使用，无需下载。

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/1822520752/badmintonrapidscoreboard.git

# 进入目录
cd badmintonrapidscoreboard

# 启动本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

## ❓ 常见问题

<details>
<summary><b>🔧 安装和使用</b></summary>

**问：需要安装什么吗？**
答：不需要！这是一个纯前端应用。只需在浏览器中打开 `index.html` 或访问在线演示即可。

**问：支持哪些浏览器？**
答：支持所有现代浏览器：Chrome（推荐）、Firefox、Safari、Edge。IE不支持。

**问：可以在手机上使用吗？**
答：当然可以！本应用采用响应式设计，完美支持手机、平板和所有设备。

**问：需要联网吗？**
答：不需要！本应用完全离线可用。所有功能无需网络连接。

</details>

<details>
<summary><b>💾 数据相关问题</b></summary>

**问：我的数据存储在哪里？**
答：数据存储在浏览器的 LocalStorage 中，仅在本地设备上，从不上传到任何服务器。

**问：我的数据会丢失吗？**
答：正常使用不会导致数据丢失。但清除浏览器缓存/数据、使用隐私/无痕模式、卸载浏览器会删除数据。建议定期备份数据。

**问：如何备份我的数据？**
答：进入"设置" → "数据管理" → "备份数据"，会下载一个JSON文件。

**问：如何恢复我的数据？**
答：进入"设置" → "数据管理" → "恢复数据"，选择之前导出的JSON文件。

</details>
