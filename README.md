# 🏸 羽毛球极速记分板（专业版）

一个**开箱即用的羽毛球比赛管理单页应用**：不仅能实时记分，还包含智能分组、费用分摊、历史记录、排行榜、能力分析与成就系统，适合球友局、俱乐部活动和日常训练。  

> 如果你希望“打开网页就能马上开打”，这个项目就是为你准备的。

## 项目简介

`badmintonrapidscoreboard` 是一个纯前端项目（单个 `index.html` 即可运行），重点解决羽毛球活动中常见的几件事：

- 比赛中：快速加减分、局数管理、计时和赛制切换。
- 比赛后：自动沉淀历史数据与统计图表。
- 组织活动时：分组与费用计算不再手工算。
- 长期使用：形成排行榜、选手能力画像和成就进度。

## 核心功能一览

- **实时记分板**：支持双方队名、比分加减、局数显示与计时。  
- **多种赛制**：支持 11/15/21 分及自定义模式。  
- **智能分组**：帮助多人活动快速生成分组并保留历史。  
- **费用计算**：活动费用分摊、导出与历史留存。  
- **历史记录 + 可视化**：自动记录比赛，并展示趋势/分布图。  
- **排行榜**：支持按胜场、参赛、胜率、时长等维度查看。  
- **选手能力分析**：查看个人基础统计、排名表现与近期状态。  
- **成就系统**：记录里程碑，增强训练与参赛反馈感。  
- **主题设置**：内置多套主题，适配不同展示场景。

## 适用场景

- 球馆临时组局（手机/平板直接打开即可用）
- 俱乐部固定活动（持续积累历史数据）
- 教练训练课（记录训练对抗结果）
- 朋友娱乐赛（减少手工记分与争议）

## 快速开始

### 方式一：直接本地打开

双击 `index.html` 即可运行。

### 方式二：本地静态服务器（推荐）

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080`
