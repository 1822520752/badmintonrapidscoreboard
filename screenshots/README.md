# 📸 项目截图说明

## 如何添加项目截图

### 方法1：手动截图（推荐）

1. **访问在线演示**
   - 打开 https://1822520752.github.io/badmintonrapidscoreboard/

2. **截取不同页面**
   - 主记分板页面
   - 智能分组页面
   - 费用计算页面
   - 历史记录页面
   - 排行榜页面
   - 能力分析页面
   - 成就系统页面
   - 设置页面

3. **截取不同主题**
   - 暗夜紫（默认）
   - 活力橙
   - 天空蓝
   - 森林绿
   - 樱花粉
   - 黑金商务

4. **截图命名规范**
   ```
   scoreboard-main.png        # 主记分板
   scoreboard-match.png       # 比赛进行中
   scoreboard-victory.png     # 胜利界面
   
   grouping-input.png         # 分组输入
   grouping-result.png        # 分组结果
   
   cost-calculator.png        # 费用计算器
   
   history-records.png        # 历史记录
   leaderboard.png            # 排行榜
   skill-analysis.png         # 能力分析
   achievements.png           # 成就系统
   settings.png               # 设置页面
   
   theme-purple.png           # 暗夜紫主题
   theme-orange.png           # 活力橙主题
   theme-blue.png             # 天空蓝主题
   theme-green.png            # 森林绿主题
   theme-pink.png             # 樱花粉主题
   theme-blackgold.png        # 黑金商务主题
   ```

5. **保存位置**
   - 将截图保存到 `screenshots/` 文件夹

### 方法2：使用浏览器开发者工具

1. 打开浏览器开发者工具（F12）
2. 切换到设备模拟模式（Ctrl+Shift+M）
3. 选择设备尺寸（推荐：iPhone 12 Pro 或 390x844）
4. 右键 → 截图 → 截取完整页面

### 方法3：使用截图工具

推荐工具：
- Windows: Snipaste, ShareX
- Mac: CleanShot X, Kap
- Linux: Flameshot, Kazam

## 截图要求

### 尺寸建议
- 宽度：390px - 428px（移动端）
- 高度：根据内容自适应
- 格式：PNG（推荐）或 JPG
- 文件大小：< 500KB（建议压缩）

### 内容建议
- 展示核心功能
- 突出UI设计亮点
- 包含实际数据（示例数据）
- 展示不同主题效果

## 更新README

截图保存后，更新README.md中的图片路径：

```markdown
![描述](./screenshots/文件名.png)
```

示例：
```markdown
![主记分板](./screenshots/scoreboard-main.png)
```

## 注意事项

1. 不要包含真实用户的个人信息
2. 使用示例数据而非真实比赛数据
3. 确保截图清晰，分辨率适中
4. 压缩图片以减小仓库体积
5. 定期更新截图以反映最新版本

## 图片压缩工具
